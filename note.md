

| 指标名称 (Key) | 中文名称 | 代码中的计算逻辑 | 核心含义 | 评估标准与解读 |
| --- | --- | --- | --- | --- |
| **`final_capital`** | **最终资金** | 循环结束后的 `cash` 余额 | 回测结束时账户的总资产（本金+盈亏）。 | **结果导向**：最直观的“最后剩多少钱”。 |
| **`cumulative_return_pct`** | **累计收益率** | `(最终资金 - 初始资金) / 初始资金` | 整个回测周期内的总回报比例。 | **越高越好**：但需结合回测时长来看。如果是10年的100%和1年的100%，含金量完全不同。 |
| **`annualized_return_pct`** | **年化收益率** |  | 将总收益折算成每年的平均复利增长率。 | **核心指标**：用于横向对比。<br>

<br>• **>20%**：优秀<br>

<br>• **>40%**：极佳（需警惕过拟合） |
| **`max_drawdown_pct`** | **最大回撤** | 历史最高点到随后最低点的最大跌幅 | **极端风险测试**：运气最差时买入并持有到最差时卖出，会亏多少。 | **越低越好**：<br>

<br>• **<10%**：稳健<br>

<br>• **>20%**：风险较高，心理压力大<br>

<br>• **>30%**：大部分资金方无法接受 |
| **`sharpe`** | **夏普比率** | `(年化收益 - 无风险利率) / 年化波动率` | **性价比**：每承受一单位波动（风险），能换来多少超额收益。 | **越高越好**：<br>

<br>• **<1**：不合格<br>

<br>• **>1.5**：良好<br>

<br>• **>2.0**：非常优秀 |
| **`win_rate_pct`** | **胜率** | `盈利的天数 / 总交易天数` | **日胜率**：在回测的这些天里，有多少天账户是赚钱的。 | **参考指标**：<br>

<br>• 趋势策略通常在 40%-50%（靠大赚小赔）<br>

<br>• 均值回归/高频通常需 >55% |
| **`profit_factor`** | **盈亏比** | `总盈利金额 / | 总亏损金额 | ` |

<br>此代码假设**每日全仓换股**（每天全卖再全买），可能高估了实际换手率和手续费成本。 |
